This repository contains an R-based web scraping script built with **rvest**, **dplyr**, and **stringr**.
It extracts the world population by country table from Worldometer, renames columns into a clean structure, converts messy text values into numeric format, appends timestamp and source information, and saves the final dataset as `worldometers_population.csv`.

The project is useful for learning:

* web scraping with R
* HTML table extraction
* data cleaning and transformation
* saving structured data for analysis

## Key Features

* Scrapes live population data from Worldometer
* Extracts and parses HTML tables
* Cleans numeric columns like percentages and comma-separated values
* Adds scrape timestamp and source URL
* Exports cleaned data to CSV
